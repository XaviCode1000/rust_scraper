You can install the extension from [Chrome store](https://chrome.google.com/webstore/detail/web-scraper/jnhgnonknehpejjnehehllkliplmbmhn) or [Firefox browser Add-ons](https://addons.mozilla.org/en-US/firefox/addon/web-scraper/). After installation, you should restart the browser to make sure the extension has fully loaded. If you don't want to restart browser then use the extension only in tabs that are created after installation.

## Requirements

Browser version requirements:

*   Chrome: `100+`
*   Firefox: `113+`
