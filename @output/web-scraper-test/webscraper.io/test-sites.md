Here are some sites that you can use for training while learning how to use the Web Scraper. 

* * *

## [E-commerce site](https://webscraper.io/test-sites/e-commerce/allinone)

E-commerce site with multiple categories, subcategories. All items are loaded in one page. 

![E-commerce site](https://webscraper.io/images/test-sites/screens/test-sites-ecommerce-allinone.png)

* * *

## [E-commerce site with pagination links](https://webscraper.io/test-sites/e-commerce/static)

E-commerce site with multiple categories, subcategories. Standard links are used for pagination. 

![E-commerce site](https://webscraper.io/images/test-sites/screens/test-sites-ecommerce-static.png)

* * *

## [E-commerce site with AJAX pagination links](https://webscraper.io/test-sites/e-commerce/ajax)

E-commerce site with multiple categories, subcategories. Dynamic links that use data without reloading the page for pagination. 

![E-commerce site](https://webscraper.io/images/test-sites/screens/test-sites-ecommerce-ajax.png)

* * *

## [E-commerce site with "Load more" buttons](https://webscraper.io/test-sites/e-commerce/more)

E-commerce site with multiple categories, subcategories. Instead of using pagination this site uses a "Load more" button to load more items. 

![E-commerce site](https://webscraper.io/images/test-sites/screens/test-sites-ecommerce-more.png)

* * *

## [E-commerce site that loads items while scrolling](https://webscraper.io/test-sites/e-commerce/scroll)

E-commerce site with multiple categories, subcategories. Instead of using pagination this site loads items when user scrolls the page down. 

![E-commerce site](https://webscraper.io/images/test-sites/screens/test-sites-ecommerce-allinone.png)

* * *

## [Table playground](https://webscraper.io/test-sites/tables)

This page contains multiple tables. You can train using Table selector here. 

![table playground](https://webscraper.io/images/test-sites/screens/test-sites-tables.png)
