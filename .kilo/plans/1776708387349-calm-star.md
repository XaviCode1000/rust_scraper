### Skills GitNexus Disponibles en `.opencode/skills/gitnexus/`
```
gitnexus-cli/            # Indexar, status, clean, wiki

gitnexus-guide/       # Recursos, herramientas, schema
gitnexus-impact-analysis/  # Análisis de blast radius (CRÍTICO)
gitnexus-refactoring/ # Renombrar, extraer, dividir código
```

### Herramientas MCP de GitNexus (usar SIEMPRE)
| Herramienta |Uso |
|------------|-----|
| `gitnexus_query` | Encontrar código por concepto (execution flows) |
| `gitnexus_context` | Vista 360° de un símbolo |
| `gitnexus_impact` | **CRÍTICO**: Análisis de blast radius antes de editar |
| `gitnexus_detect_changes` | Pre-commit scope check |
| `gitnexus_rename` | Renombrar seguro |
| `gitnexus_cypher` | Consultas personalizadas al graph |

### Flujo GitNexus Obligatorio para Agentes
```
1. Antes de editar CUALQUIER símbolo:
   → gitnexus_impact({target: "symbol_name", direction: "upstream"})
   
2. Para entender código:
   → gitnexus_query({query: "concept"}) 
   → gitnexus_context({name: "function_name"})
   
3. Antes de commit:
   → gitnexus_detect_changes({scope: "staged"})
   
4. Para debugging:
   → gitnexus_query({query: "<error>"})
   → gitnexus_context({name: "suspect_function"})
```
